# AI-Based OSINT Analysis and Monitoring System

An AI-powered Open Source Intelligence (OSINT) platform designed for real-time news monitoring, sentiment analysis, trend tracking, and geopolitical event analysis.

## Features

- Real-time news aggregation
- Sentiment analysis using NLP
- Interactive data visualization
- Topic and trend monitoring
- Geopolitical intelligence dashboard
- FastAPI backend
- React frontend

## Tech Stack

### Backend

- Python
- FastAPI
- TextBlob
- Pandas

### Frontend

- React
- Vite
- Axios
- Chart.js
